## 核心要点

### LangChain 方式
- 基于 Chain 的线性执行
- `AgentExecutor` 封装 ReAct 循环
- 适合简单、顺序的 Agent 任务

### LangGraph 方式
- 基于 Graph 的状态机编排
- 显式的 State + Node + Edge 定义
- 支持循环、条件分支、并行执行
- 适合复杂 Multi-Agent 编排
